Connected Scatterplot Study

In this study, you will be presented with a number of line drawings that we ask you to reproduce. Each drawing will consist of a single line and represent the comparison of two stock prices over time. Each stock is represented by one axis, with the line connecting the points in temporal order.

  1. The line will appear for one second.
  2. After it disappears, trace it from memory as best as you can.
  3. Click Done to see how you did.
  4. Click Next to go to the next one.

Thank you for participating!
